# Fancy Door Anim Rebuild

Smooth, animated door opening for Fabric 1.21.11.

![Open Door Animation](opendoor.gif)

## Features
- Smooth door open/close animation
- Lightweight and client-friendly
- Works with vanilla doors

## Installation
1. Install Fabric Loader for Minecraft 1.21.11
2. Install Fabric API
3. Drop the mod JAR into your mods folder

## Build
- Run `gradlew build`
- The JAR will be in `build/libs`
